# Example Incident Report: Phishing Email

## 1. General Information
- Date: 2026-03-24
- Time: 10:30
- Reported by: End User
- Incident ID: INC-001

## 2. Incident Description
A user reported receiving a suspicious email asking them to verify their account.

## 3. Affected Systems
- User workstation

## 4. Detection Method
- User Report

## 5. Indicators of Compromise (IOCs)
- URL: http://fake-bank-login.com
- Keyword: "verify your account"

## 6. Analysis
The email attempted to trick the user into entering credentials on a fake login page.

## 7. Impact Assessment
- No credentials entered
- No system compromise detected

## 8. Response Actions
- Suspicious URL blocked
- User instructed to delete the email
- Security awareness reminder sent

## 9. Recommendations
- Improve email filtering
- Continue phishing awareness training

## 10. Status
Resolved
